侃哥的网页朗读神器

朗读软件很多，能方便快捷的读公众号、读新闻客户端的却很少，本APP解决了这些痛点

  1. 保持原排版的选段朗读功能，可通过点击跳过前后的广告
  2. 自动滚屏（不支持自动翻页），解决系统内置朗读工具只读当前页的弊端
  3. 离线语音合成，不耗流量

1. 安装 语音合成(TTS) 引擎

可以通过应用商店安装TTS引擎，目前推荐的中文TTS引擎是科大讯飞的【讯飞语记】，讯飞语记可以选不同的发音人。

2. 设置默认引擎

设置首选引擎为【讯飞语记】，设置成其它不支持中文的引擎时，遇到中文时会无声音。部分机型可能出现【科大讯飞语音引擎】和【讯飞语记】两个选项，请选择【讯飞语记】

由于不同厂家定制ROM的不同，如果跳转失败，请手工到【设置->语言和输入->文字转语音输出】中进行设置

3. 使用帮助

操作设置分为两大区：

  • 浮动窗口：朗读设置
    1. 播放
    2. 暂停
    3. 设置：语速设置
  • 右上角选项菜单：浏览器设置和其它功能
    1. 设置网址或文本：相当于浏览器的地址栏功能
    2. 禁止加载图片：可节省流量
    3. 去评价：跳到应用商店

打开网页后，点击“播放”，即播放当前网页，可通过触屏点击快速跳转到其它段落

当前段会自动高亮

选段高亮后，再点“播放”，立即朗读当前段，否则等前一段读完后接下来会读当前段的下一段

4. 打开网页

设置网址或文本（本APP右上角）：可输入网址，也可以输入文本。如果是网址，请编辑成正确地址，点击“网址浏览”，反之点击“文本显示”


发送到（其它APP）：在其它软件的“分享”中选择“神器”，在弹出的对话框中去除多余的文字，只保留网址，点击“网址浏览”。如：网易新闻客户端

在浏览器打开（其它APP）：在其它软件的中“在浏览器打开”中选择“神器”，在弹出的对话框后，点击“网址浏览”。如：微信

分享至邮件（其它APP）：部分软件只有分享到微信或邮件的功能，可以选择邮件后，再选“神器”，在弹出的对话框后，点击“文本显示”(部分APP邮件分享只会提供摘要)。如：万得APP

复制链接（其它APP）：由于“在浏览器中打开”时不小心选过“总是”，导致没有机会再选择“神器”，只好复制链接

复制文本（任意APP）：部分网页无法朗读，如使用iframe技术的网页无法跨域。解决的办法是网页中长按，弹出菜单后全选复制，粘贴到“设置网址或文本”，点击“文本显示”即可。如：百度移动版网页

6. 微信“在浏览器打开”

微信的默认浏览器功能并不是使用系统设置，而是自己实现的，卸载指定的默认APP才能再次弹出打开方式对话框，装回再次恢复，只能用其它浏览器设成默认，然后删除来替换
